# 🐍 Snake Water Gun

A simple Snake Water Gun game made using Python.

## Features

- Play against the computer
- Multiple rounds
- Random computer choices
- Score tracking
- Winner announcement

## Controls

- `s` → Snake
- `w` → Water
- `g` → Gun

## Rules
- Snake win against Water
- Water win against Gun
- Gun win against Snake
